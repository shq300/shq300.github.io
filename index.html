RPG INVENTORY AND INTERACTION SYSTEM
FOR UNREAL ENGINE 4 & 5

Multiplayer Inventory and Interaction System made in 100% using Blueprints. Designed to be intuitive, flexible and expandable. Try the Playable Demo now!   MARKETPLACE PAGE

SYSTEM FEATURES:

Inventory System Managed inside of a Component, that can be added to any actor. Contains lot of useful and customisable options.
Interaction System Player Interaction with various pre-defined objects and ability to make advanced 'puzzle' systems for your game.
Crafting System Versatile Crafting System that can handle multiple Crafting Stations and provides many parameters that can be tweaked for your project specifics.
Stat & Buff System
Simple stat component, that can be added to any actor and a buff system to use with consumables.
Leveling & Experience System Leveling Component handles Level and Experience of the Player.
Multiplayer Ready System is already Network Replicated, both Client as Server and Dedicated Server.
