Tutorial: Adding Separate Crafting Station Window

Made using RPG I&I v1.9
Tutorial Result Example New Alchemy Station with separate Crafting Window
1. Open E_CraftingStation, add your new Station Name to it
2. Duplicate the BP_ForgeStation, re-name it then select CraftingCore Component, change the Crafting Station parameter to 'Alchemy' and the Crafting Data Table you wanna put in there (in this case Alchemy Station will use the DT_CraftingListConsumables)
3. Duplicate the WB_ForgeStationWindow, rename it then change it's layout to your own preference
4. Return Back to the Crafting Station BP (in this case BP_AlchemyStation), on Event Start Interaction -> Open Crafting Widget node, swap previous widget class with your newly created one (in this case WB_AlchemyStationWindow)
