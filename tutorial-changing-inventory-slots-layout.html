Tutorial: Changing Inventory Slots Layout

Made using RPG I&I v1.9
Tutorial Result Example Changed Inventory Slots Row Lenght + Slot Size and Padding
1. Open WB_PlayerInventory, change Row Lenght to a new value. In this example Row Lenght = 4
2. Go to Designer Tab, select the Uniform Grids you want to change and set new Slot Padding + Min Desired Slot Size
