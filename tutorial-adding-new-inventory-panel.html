Tutorial: Adding New Inventory Panel

Made using RPG I&I v1.9
There's a bug in Unreal 5.1, which can crash the Editor. To prevent it do NOT re-order the variables after adding them inside AC_InventoryCore.
Tutorial Result Example
1. Open E_InventoryPanels, add new enum value = P5
2. Open AC_InventoryCore, duplicate InventoryP4 variable (array), re-name it to 'InventoryP5'. Create new variable: Integer, 'InventorySizeP5'. Mark it as Editable and Set Replication to 'RepNotify'
3. Go to newly created 'On Rep Inventory Size P5' function and add the same nodes as in the image
4. Go to 'Apply Changes to Inventory Array' and add new connection to 'Inventory P5' array
5. Add highlighted nodes inside 'Get Combined Inventories'
6. Add InventoryP5 and InventorySizeP5 inside 'Get Inventory and Size'
7. Open AC_PlayerInventory / Class Defaults, add P5 to the Panels to Use. Make sure the Inventory Size P5 is set properly
8. Open WB_PlayerInventory, duplicate ButtonPanel and UniformGrid, re-name them. Change Button Panel to P5 and Style ON/OFF
9. Duplicate Create Inventory event, re-name it to 'Create Inventory P5'. Swap Uniform Grid to Inventory UG5
10. On Event Construct add the newly created event 'Create Inventory P5'
11. On Event Set Active Panel pin ButtonPanel5
12. On Event Refresh Inventory add highlighted node on P5 switch
13. Inside Get Active Panel add a check for Inventory UG5
14. Inside Get Uniform Grid from Panel pin Inventory UG5
15. Inside Switch Inventory Panel pin Button Panel 5
16. Inside Switch To Next Inventory Panel, re-order and add highlighted nodes
17. Inside Set Panel Title, promote P5 to a new (editable) variable and change the text value
18. Now make sure each Item that suppose to go inside that panel has it assigned inside their Item Class
