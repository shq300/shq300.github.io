Tutorial: Adding New Equipment Slot

Made using RPG I&I v1.9
Tutorial Result Example Added New Equipment Slot + Player Attachment (to Socket or Character Skeleton)
1. Open E_ItemSlot, add new enumerator (your new Slot name, in this example = 'Bow')
2. To prevent the Unreal Structure bug do the following:
I. Save the project and re-open it
II. Right Click on the RPG Inventory folder and select Resave All
 III. Re-open the project once again
3. Open WB_Equipment, duplicate one of the Slots, re-name it and change the Equipment Slot enumeration to the newly created one
4. Go to 'Get Widget Slot by Item Slot' function, pin your new slot to the Select node
5. You can now assign this Slot to the Items in the Items Data Tables
After that depending on the Equipment Slot you may want to Attach the equipped item to the Socket(by spawning Equippable Class) or directly to the Skeleton: Attaching Item to the Socket (Equippable Class, mostly Weapons)
1A. Open AC_PlayerEquipment / Attach Item to Equipment, connect the Slot to 'Server Spawn Equippable Actor' node
2A. Go to 'Detach Item from Equipment' and connect your Slot to the 'Server Destroy Equippable Actor' node
3A. Go to 'Spawn and Attach Equippable Actor', duplicate one of the Spawn & Attach nodes. Connect the new ones to your Slot (in this example I've also marked Combat Mode to False, so that the Item will be attached to the Non-Combat Socket by default)
4A. Open BP_PlayerCapture / Attach Item, do the same thing as previously - duplicate the nodes and connect to your Slot from the Switch node (here I've also marked Combat Mode to False)
5A. Go to Detach Item function, pin your Slot to the Detach Equippable (Destroy Actor) nodes
6A. Open your Character Skeleton and add new socket to it, in this example I've added a Non-Combat socket named 'Bow'
7A. Add second socket for the Combat Mode, in this case 'bow_use'
8A. Create a new child of BP_BaseEquippable, set the Static or Skeletal Mesh. Select Class Defaults and assign the previously created Non-Combat and Combat sockets for the specified Slot
9A. At the end add the Equippable Class to your Item Class
Attaching Item to the Character Skeleton (Skeletal Mesh reference, mostly Armor)
1B. Open AC_PlayerEquipment / Attach Item to Equipment, connect your Slot to the 'Server Update Equipment Mesh' node
2B. Go to 'Detach Item from Equipment', connect your Slot to the Server Update Equipment Mesh node
3B. Open the RPG Inventory Character, duplicate one of the SkeletalMeshes, re-name it and make sure the Transform has the default values
4B. Create new variable: of type 'Skeletal Mesh', re-name it, change the Category to Equipment Meshes, set Replication to 'RepNotify'. Connect the Update Equipment Mesh Event to it
5B. Open the newly generated On Rep function. Add the highlighted nodes in there, make sure to swap the variable references with your own
6B. Open BP_PlayerCapture, duplicate one of the Skeletal Meshes and re-name it
7B. Go to Attach Item function, connect your Slot to the Armor Attachment nodes
8B. Pin your newly created SkeletalMesh to the selection node inside Attach Item function
9B. Go to Detach Item function, duplicate one of the Armor detachment nodes, then connect your Skeletal Mesh to it
