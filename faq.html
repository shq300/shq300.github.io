Frequently Asked Questions (FAQs)

Can I test this asset before I purchase?
Sure, you can test this asset through a playable demo available here
Where can I find more learning resources about the system?
More resources (tutorials, additional files) can be found in our Discord Server, after the verification you will receive a 'verified' role and gain access to the private channels.
Are the previous versions of the system still available for download?
Yes, you can still download previous versions of the system either by creating a new project using specific Unreal Engine version or by getting the link from the Discord channel (requires becoming a verified member of the server).
Where can I find the system roadmap containing future plans?
System roadmap is available here .
After modifying a structure I am getting some errors, is there a way to fix it?
If it's the refreshment issue, you should be able to fix it by Right Clicking on a Content folder and selecting Resave All. After that re-open your project. All of the nodes should refresh themselves and the errors should no longer appear. Additionaly you can look for potential errors in the OutputLog.
